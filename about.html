About This Showcase

This portfolio showcases UI/UX components designed and developed for Booxi's appointment booking platform. Each component demonstrates attention to detail, accessibility, and modern design principles.

Built with a focus on creating intuitive, beautiful interfaces that enhance the user experience. The components leverage Figma MCP (Model Context Protocol) integration with Cursor for streamlined design-to-code workflows.

All components are production-ready, fully responsive, and built with performance and accessibility in mind.
