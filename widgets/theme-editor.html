Overview

The Booking Widget Theme Editor provides a powerful yet intuitive interface for customizing the appearance of booking widgets. Users can modify colors, typography, spacing, and other visual properties with instant preview of changes across all widget states.

Key Features

  • Live Preview: See changes instantly in the widget preview
  • Color Pickers: Full color customization with hex, RGB, and presets
  • Typography Controls: Font family, size, and weight selection
  • Spacing Editor: Adjust padding and margins with visual feedback
  • Border Radius: Control corner roundness for modern or traditional looks
  • Preset Themes: Quick start with pre-designed theme templates
  • Export/Import: Save and share theme configurations
  • Dark Mode Toggle: Test widget appearance in different modes

Use Cases

  • Brand customization for enterprise clients
  • White-label booking solutions
  • A/B testing different visual styles
  • Seasonal theme variations
