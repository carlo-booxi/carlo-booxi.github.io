Overview

A comprehensive status bar navigation system designed for different office contexts within enterprise booking management systems. This component provides essential information and quick access to key functionality while maintaining a clean, professional aesthetic.

Key Features

  • Multiple Variants: Head Office, Back Office, and CS Dashboard themes
  • Real-time Clock: Automatically updates time and date display
  • Interactive Search: Click-to-activate search with keyboard support
  • Dropdown Menus: Language selection and account management
  • Contextual Tooltips: Helpful labels on hover for all actions
  • Notifications: Alert system with visual indicators
  • Responsive Layout: Adapts to different screen sizes
  • Keyboard Navigation: Full keyboard accessibility

Variants

  • Head Office: Dark theme for corporate management interface
  • Back Office: Light theme for store-level operations
  • CS Dashboard: Simplified layout for customer service teams
